## Formula 2: Numbered Cross Patterns

**Pattern**: Numbered crosses appear at 16-position intervals with type-specific gate arrangements

### Explanation

Crosses with numerical suffixes (2, 3, 4) represent the **same archetypal energy** appearing at different positions around the wheel, spaced exactly 16 positions apart.

**Key relationships:**
- 16 positions = 64 ÷ 4 = **quarter wheel**
- 16 positions = **90 degrees** (360° ÷ 4)
- All instances in a numbered series must have the **same cross type**
- Gate values follow distinct patterns based on cross type

### Mathematical Structure

```
Cross family "X":
  X 2:  Position N
  X 3:  Position N + 16  (90° later)
  X 4:  Position N + 32  (180° later)
  X 5*: Position N + 48  (270° later) - if it existed
  (*X 1 would be at N - 16)
```

### Why 16 Positions?

The wheel has 64 gates. Dividing by 4 creates natural quarters:
- 64 ÷ 4 = **16 gates per quarter**
- Each quarter represents 90° of the 360° wheel
- Numbered crosses mark these quarter boundaries

### Gate Value Patterns

#### LAX Numbered Pattern: Mirror Symmetry

For LAX numbered series, base and "_2" variant have **mirrored gate values**:
- **PS ↔ PE swapped**
- **DS ↔ DE swapped**

**Example: Clarion Family (all LAX)**
- Clarion:   PS=51, PE=57, DS=61, DE=62
- Clarion 2: PS=57, PE=51, DS=62, DE=61 ✓ (mirrored)

**Example: Education Family (all LAX)**
- Education:   PS=12, PE=11, DS=25, DE=46
- Education 2: PS=11, PE=12, DS=46, DE=25 ✓ (mirrored)

#### RAX Numbered Pattern: Gate Rotation

For RAX numbered series, the four gates **rotate through all positions** (2, 3, 4):

**Example: Eden Family (all RAX)**
- Eden:   PS=36, PE=6,  DS=11, DE=12
- Eden 2: PS=12, PE=11, DS=36, DE=6  (rotation)
- Eden 3: PS=6,  PE=36, DS=12, DE=11 (rotation)
- Eden 4: PS=11, PE=12, DS=6,  DE=36 (rotation)

The four gates (36, 6, 11, 12) cycle through all four positions.
