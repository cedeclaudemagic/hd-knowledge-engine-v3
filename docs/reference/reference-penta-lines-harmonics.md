## Electromagnetic Translation

In electromagnetic terms, harmonics are integer multiples of a fundamental frequency. When a string vibrates, it produces not only its fundamental frequency but all its harmonic overtones simultaneously. Certain combinations create consonance (pleasing resonance), others create dissonance (clash).

Fourier's theorem (early 1800s) proved that ANY complex waveform can be decomposed into a series of simple sine waves at harmonic frequencies. This is the mathematical foundation of all signal processing.

Ra's harmonic pairs map directly:

- **Lines 1, 2, 3** = Input characteristics (how the circuit receives)
- **Lines 4, 5, 6** = Output characteristics (how the circuit transmits)

When you activate one line, you automatically invoke its harmonic partner. They are electromagnetically inseparable—like fundamental and overtone.

## Yin/Yang as Impedance Characteristics

Each hexagram line is either:

- **Yang (solid line ———):** Active, positive, conductive
- **Yin (broken line — —):** Passive, negative, resistive

In electromagnetic terms:

- **Yang:** Low impedance, high conductivity, active transmission
- **Yin:** High impedance, low conductivity, passive reception

POSITION tells you the frequency harmonic (which function). YIN/YANG tells you the impedance characteristic (how that function behaves).

With 6 lines, each either yin or yang, there are 2⁶ = 64 possible combinations—the 64 hexagrams. Each gate can be activated with any of 64 line configurations, creating different impedance signatures for the same circuit function.
